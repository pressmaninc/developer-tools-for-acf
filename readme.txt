Provide developer tools for ACF.

== Description ==
This plugin adds 2 functionalities for developers who use ACF.
1. Add a display of field details in ACF form.
2. Add extra columns in ACF field group list.

For details, see this plugin's setting page.

== Installation ==

1. Upload the 'developer-tools-for-acf' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
